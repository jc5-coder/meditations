# 沉思录 · Meditations

> 万物皆流 — 选一个词，遇见一位哲人

**沉思录**是一个交互式哲学名言词云网站。150+ 个词如河水般流动，鼠标靠近词会放大高亮，轻轻一点，就有一段哲人的故事在等你。

🌐 **在线体验**: https://jc5-coder.github.io/meditations  
📦 **GitHub**: https://github.com/jc5-coder/meditations

---

## ✨ 创意理念

我们生活在一个信息过载的时代，但内心却越来越安静——不是平静，而是安静的焦虑。

沉思录的灵感来自一个很简单的想法：**当你觉得焦虑、迷茫、无聊的时候，打开这个页面，让词语从眼前流过。碰到哪个词，就是哪个词在找你。**

赫拉克利特说"万物皆流"。哲学不是摆在书架上的经典，而是活在每一个当下的选择里。你此刻碰到的词，也许就是你此刻需要的答案。

## 🎨 产品特色

### 流动的哲学
- **Fibonacci 球体分布** — 150+ 个词均匀分布在球面上，如星辰流转
- **鼠标交互高亮** — 鼠标靠近的词放大 1.8 倍、金色发光，像在回应你
- **点击涟漪动效** — 点击时出现金色光晕，确认感满满

### 有温度的内容
- **29 位哲人** — 从古希腊到明代，从中国诸子百家到西方哲学，涵盖东西方思想脉络
- **掌故 + 省思** — 不是冷冰冰的词条，而是一段真实的故事 + 一段与现代人的共鸣
- **四语言支持** — 中文 / English / 日本語 / Ελληνικά
- **296 个关键词** — 每个词都指向一位哲人的故事，点击即达

### 精致的视觉
- **雅典学院背景** — 拉斐尔名画做底层，低透明度羊皮纸质感
- **哲思之门设计** — 左右门框 + 顶部门楣，古希腊文 Φιλοσοφία / Σοφία
- **暖金色点缀** — 全局统一暖色调，避免 AI 常见的紫蓝霓虹感

- **React 19** + **Vite 8** + **Tailwind CSS 4**
